# Rielcode Laravel

Portfolio + agency commerce site. Laravel 12, Filament admin, MySQL, AI chatbot.

## Requirements

- PHP 8.2+
- Composer
- Node.js 18+
- MySQL (XAMPP works)

## Setup (new device)

```bash
# 1. Clone
git clone <repo-url>
cd rielcode-laravel

# 2. Copy env and fill in values
cp .env.example .env

# 3. Edit .env — set these at minimum:
#    DB_DATABASE=rielcode
#    DB_USERNAME=root
#    DB_PASSWORD=        (blank for XAMPP default)
#    OPENAI_API_KEY=sk-...

# 4. Run setup (installs deps, generates key, migrates, builds assets)
composer run setup

# 5. Start dev environment
composer run dev
```

App runs at http://127.0.0.1:8000  
Admin panel at http://127.0.0.1:8000/admin

## Environment Variables

| Key | Description |
|-----|-------------|
| `APP_KEY` | Auto-generated by `key:generate` |
| `DB_DATABASE` | MySQL database name (`rielcode`) |
| `DB_USERNAME` | MySQL user (`root` for XAMPP) |
| `DB_PASSWORD` | MySQL password (blank for XAMPP default) |
| `OPENAI_API_KEY` | OpenAI API key for AI chatbot |
| `OPENAI_MODEL` | Model to use (default: `gpt-4o-mini`) |
| `APP_URL` | Base URL (`http://127.0.0.1:8000` locally) |
| `PROGRESS_URL` | Same as APP_URL locally |
| `BRIEF_URL` | Same as APP_URL locally |
| `TESTIMONIAL_URL` | Same as APP_URL locally |

## Commands

```bash
composer run setup    # First-time setup
composer run dev      # Start all dev processes (server + queue + logs + Vite)
php artisan migrate   # Run migrations
npm run build         # Build assets for production
./vendor/bin/pint     # Lint PHP
php artisan test      # Run tests
```

## Database

MySQL. Create the database before running setup:

```sql
CREATE DATABASE rielcode CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

XAMPP: start Apache + MySQL, then run setup.
